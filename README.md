# NNAK Membership Management System

A premium, high-density membership management system designed for the National Nurses Association of Kenya (NNAK). This system streamlines member registration, profile management, professional development tracking (CPD), and financial ledger management.

- **CPD Points System**: Specialized ledger for tracking and awarding Continuing Professional Development points.
